# Programming Languages

Python, OCaml, and Rust projects exploring functional programming, automata, parsing, interpretation, and text processing.

**Technologies:** Python · OCaml · Dune · Rust · Cargo

## Highlights

- Finite-state machines, regular expressions, and nondeterministic automata.
- Lexer, parser, evaluator, and interpreter components in OCaml.
- Rust command-line and text-processing exercises.

## Getting started

1. Use Python, OCaml/Dune, or Rust/Cargo according to the selected project.
2. Run commands from the individual project directory.
3. Each project preserves its own language-specific build layout where available.

## Portfolio note

Projects are independent language-learning artifacts rather than one cross-language application.
