## Test and fixture policy

Directories explicitly identified as public, student, instructor, or autograder tests
were removed from the portfolio copy. Generic tests and fixtures remain because their
ownership cannot be determined from directory names alone.

Review every retained test, fixture, and test-data file before public release. See
`REMOVED_COURSE_TESTS.md` and `OWNERSHIP_REVIEW.md`.
